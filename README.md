# J.Lawrence Bathrooms and Kitchens Premium Site

Premium static Next.js outreach site for J.Lawrence Bathrooms and Kitchens in Basildon.

## Commands

```bash
npm install
npm run build
NEXT_PUBLIC_BASE_PATH=/j-lawrence-bathrooms-kitchens-premium-next NEXT_PUBLIC_SITE_URL=https://deanooooooooo.github.io npm run build
```

## Notes

- Built as a GitHub Pages project site with static export.
- No Google Maps embed and no testimonials/reviews.
- No public phone or email was verified; the enquiry helper prepares a Facebook message.
